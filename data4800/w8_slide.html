Introducing Convolutional Neural Networks (CNNs)

Why Called "Convolutional"?

"Convolution" = sliding a small filter over an image to detect patterns

Think of it like using a magnifying glass that slides across a page, examining each small section to find specific features!

Traditional Neural Networks

‚Ä¢ Fully connected layers

‚Ä¢ All neurons connect to all others

‚Ä¢ Massive number of parameters

‚Ä¢ Lose spatial information

‚Ä¢ Poor for images

Convolutional Neural Networks

‚Ä¢ Local connections

‚Ä¢ Shared parameters (filters)

‚Ä¢ Preserve spatial structure

‚Ä¢ Translation invariant

‚Ä¢ Designed for images

Convolution Operation: How It Works

What is a Kernel/Filter?

‚Ä¢ Small matrix (usually 3√ó3, 5√ó5)

‚Ä¢ Contains learnable weights

‚Ä¢ Acts as a "feature detector"

‚Ä¢ Examples: edge detector, blur, sharpen

What are Feature Maps?

‚Ä¢ Output after applying kernel to input

‚Ä¢ Each map detects different features

‚Ä¢ Preserve spatial relationships

‚Ä¢ Become input for next layer

Why kernels matter: They automatically learn to detect important visual patterns like edges, textures, and shapes - no manual programming required!

CNN Components in Detail

Convolutional Layer

‚Ä¢ Applies filters/kernels to extract features

‚Ä¢ Preserves spatial relationships

‚Ä¢ Learns edge, texture, and pattern detectors

Activation Layer (ReLU)

‚Ä¢ Introduces non-linearity

‚Ä¢ ReLU: f(x) = max(0, x)

‚Ä¢ Helps network learn complex patterns

Pooling Layer

‚Ä¢ Reduces spatial dimensions

‚Ä¢ Max pooling: takes maximum value

‚Ä¢ Provides translation invariance

Fully Connected Layer

‚Ä¢ Traditional neural network layer

‚Ä¢ Used for final classification

‚Ä¢ Combines all learned features
