Python Basics: Variables & Data Types

Variables: Storing Information

Think of variables as labeled boxes that hold information:

# Creating variables (no need to declare type!)
student_name = "Alice"           # String (text)
age = 24                          # Integer (whole number)
gpa = 3.85                        # Float (decimal number)
is_enrolled = True                # Boolean (True/False)

print(student_name)  # Output: Alice
print(age)           # Output: 24
Type Example Use Case
str (String) "Customer ID" Names, categories, text data
int (Integer) 100 Count of transactions, quantities
float (Float) 99.99 Prices, percentages, measurements
bool (Boolean) True or False Yes/No flags, status indicators

Python Basics: Lists & Dictionaries

Lists: Ordered Collections

# List of products
products = ["Laptop", "Mouse",
            "Keyboard", "Monitor"]

# Access items by index
print(products[0])  # Laptop
print(products[2])  # Keyboard

# Add items
products.append("Webcam")

# Get list length
print(len(products))  # 5

Dictionaries: Key-Value Pairs

# Customer information
customer = {
    "id": "CUST001",
    "name": "Alice Johnson",
    "age": 24,
    "purchases": 5,
    "is_member": True
}

# Access values by key
print(customer["name"])
# Output: Alice Johnson

# Add or update values
customer["email"] =
    "alice@example.com"

When to Use Each: Lists for ordered data (time series, rankings). Dictionaries for structured data with labels (customer records).

Python Basics: Control Flow

If Statements: Making Decisions

total_spent = 1500
is_member = True

if total_spent > 1000 and is_member:
    discount = 0.20  # 20% off
    print("Premium: 20% discount")
elif total_spent > 500:
    discount = 0.10  # 10% off
    print("Valued: 10% discount")
else:
    discount = 0
    print("Standard pricing")

For Loop: Iterate Over Sequence

# Calculate total revenue
sales = [100, 250, 175, 300, 425]
total = 0

for sale in sales:
    total += sale

print(f"Total revenue: ${total}")
# Output: Total revenue: $1250

List Comprehensions (Python Superpower!)

original_prices = [100, 200, 150, 300]

# Pythonic way (list comprehension)
markup_prices = [price * 1.1 for price in original_prices]
print(markup_prices)  # [110.0, 220.0, 165.0, 330.0]

Python Basics: Functions

Functions: Reusable Code Blocks

# Define a function
def calculate_discount(price, discount_rate):
    """Calculate discounted price."""
    discount_amount = price * discount_rate
    final_price = price - discount_amount
    return final_price

# Use the function
original = 100
discounted = calculate_discount(original, 0.15)
print(f"${original} with 15% off = ${discounted}")
# Output: $100 with 15% off = $85.0

Function Benefits

Reusability • Organization • Easier Testing • Better Collaboration

What is Artificial Intelligence?

Definition

AI is the simulation of human intelligence in machines that are programmed to think, learn, and make decisions like humans.

Key Capabilities

  • Learn from experience
  • Adapt to new inputs
  • Perform human-like tasks
  • Make decisions based on data
  • Improve over time

Examples You See Daily

  • Product recommendations (Amazon)
  • Voice assistants (Siri, Alexa)
  • Social media feeds
  • Spam filters
  • GPS route optimization
  • Netflix/Spotify recommendations

The AI Family Tree

AI ⊃ Machine Learning ⊃ Deep Learning

1. Artificial Intelligence (Broadest)

Includes: Machine Learning, Expert Systems, Computer Vision, NLP, Robotics, Speech Recognition

Example: Chess program with pre-programmed rules

2. Machine Learning (Subset of AI)

Definition: Systems that learn from data without explicit programming

Includes: Supervised, Unsupervised, Reinforcement Learning, Deep Learning

Example: Email spam filter learning from examples

3. Deep Learning (Subset of ML)

Definition: ML using neural networks with multiple layers

Includes: CNNs (images), RNNs (sequences), Transformers (language models)

Examples: ChatGPT, Image recognition, Self-driving cars

Narrow AI (ANI)

Current State: EXISTS TODAY

Capability: Specialized in ONE task

Performance: Often exceeds humans in that specific task

Examples: Chess programs, Image recognition, Language translation, Spam filters, Recommendation systems

General AI (AGI)

Current State: DOES NOT EXIST YET

Capability: Human-level intelligence across ALL domains

Performance: Can learn and apply knowledge to any problem

Examples: Sci-fi AI like Jarvis (Iron Man), AI that can write code, paint art, AND drive a car

Super AI (ASI) - Artificial Super Intelligence

Current State: THEORETICAL

Capability: Surpasses human intelligence in ALL aspects including creativity and wisdom

Examples: AI from movies like "Her" or "Ex Machina" - Purely speculative

This Course Focuses On: Narrow AI (ANI) - building specialized AI systems for business problems

How Does Machine Learning Work?

Traditional Programming

Inputs + Rules → Outputs

IF customer_spent > $1000
    THEN loyalty_tier = "Gold"
IF customer_spent > $500
    THEN loyalty_tier = "Silver"
ELSE loyalty_tier = "Bronze"

Problem: What if rules don't capture complex patterns?

Machine Learning

Inputs + Outputs → Rules (Learned automatically!)

Give the computer:

  • Customer data (age, purchases, behavior)
  • Labels (Gold/Silver/Bronze)

ML discovers: "Gold customers typically have 5+ purchases, spend >$1000, and visit weekly"

Advantage: Finds patterns humans might miss!

The Machine Learning Process

7-Step ML Workflow

  1. COLLECT DATA - Gather historical examples
  2. PREPARE DATA - Clean, format, and organize
  3. CHOOSE ALGORITHM - Select appropriate ML technique
  4. TRAIN MODEL - Let algorithm learn patterns from data
  5. EVALUATE - Test accuracy on new data
  6. DEPLOY - Use model to make predictions
  7. MONITOR & IMPROVE - Track performance, retrain as needed

We'll do this entire process starting Week 2!

Types of Machine Learning

1. Supervised Learning

Definition: Learning from labeled examples

Training Data: Inputs + Correct Answers

Example: Email Classification - 1000 emails labeled "spam" or "not spam"

Applications: Customer churn prediction, Credit scoring, Price prediction, Sales forecasting

2. Unsupervised Learning

Definition: Finding patterns in unlabeled data

Training Data: Inputs only (no labels)

Example: Customer Segmentation - Algorithm finds natural groups

Applications: Market segmentation, Anomaly detection, Recommendation systems

3. Reinforcement Learning

Definition: Learning through trial and error with rewards/penalties

Example: Game Playing AI - Try millions of games, learn winning strategies

Applications: Autonomous vehicles, Dynamic pricing, Resource allocation, Robotics

AI Ethics & Responsible AI

1. Fairness

Treat all groups equitably, avoid discriminatory outcomes

Example: Hiring AI shouldn't favor one gender

2. Transparency

Explain AI decisions, disclose when AI is being used

Example: Loan denial requires explanation

3. Privacy

Protect personal information, use minimum necessary data

Example: Anonymize customer data

4. Accountability

Humans are responsible for AI outcomes, monitor systems continuously, have human oversight

Example: Medical diagnosis AI needs doctor review

5. Safety

Test thoroughly before deployment, have fail-safes for errors, consider unintended consequences

Example: Autonomous vehicles prioritize safety
