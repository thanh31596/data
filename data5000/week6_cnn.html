What Are Convolutional Neural Networks?

CNNs are specialized AI systems that scan images in small patches to detect patterns—like edges, shapes, and objects.

How Humans See Images

We instantly recognize entire objects
Our brain processes holistically

How CNNs See Images

They scan patch by patch
Build understanding from simple to complex features
Think of a CNN as a sliding magnifying glass that examines every part of an image to find specific patterns

How CNNs Work: Three Essential Steps

1. DETECT

Convolutional Layer

Finds edges, textures, and basic patterns
→

2. SIMPLIFY

Pooling Layer

Keeps important features, reduces noise
→

3. DECIDE

Fully-Connected Layer

Combines features to make final classification

Detective Analogy

Step 1: Collect evidence (finding patterns)
Step 2: Focus on key clues (reducing noise)
Step 3: Make the verdict (classification)

Multiple Layers Deep CNNs stack these steps 10-100+ times to detect increasingly complex patterns

YOLOv8: State-of-the-Art Object Detection

"You Only Look Once" - A single pass through the image detects all objects

Traditional Methods

1. Propose regions of interest

2. Classify each region separately

3. Refine bounding boxes

5-10 FPS Processing speed

YOLO Approach

1. Divide image into grid

2. Predict all objects simultaneously

3. Filter and refine predictions

60+ FPS Real-time processing

YOLOv8 Capabilities

  • Detection: Locate and classify objects in images
  • Segmentation: Identify exact pixel-level object boundaries
  • Tracking: Follow objects across video frames
  • Classification: Categorize entire images

Dataset Overview

Brain Tumor MRI Images: 44 different tumor types and imaging modalities

Imaging Modalities

• T1-weighted imaging
• T2-weighted imaging
• T1C+ (contrast-enhanced)

5,000+ Total MRI images in dataset

Tumor Types Include

• Meningioma (most common)
• Glioblastoma
• Astrocytoma

44 Different classification categories
Challenge: Class imbalance—some tumor types have 300+ images, others have only 20-50. This affects model training and requires careful dataset balancing.

Using Roboflow for Medical Classification

Platform Overview

Roboflow handles the complex CNN infrastructure so you can focus on the business application

Step 1: Dataset Preparation

Organize MRI images by tumor type on your computer. Balance classes by selecting equal numbers of images per category.

Tip: Use 80% of smallest class for training (e.g., if smallest class has 50 images, use 40 per class)

Step 2: Upload to Roboflow

Create project → Upload organized folders → Roboflow automatically labels images based on folder names

1,600 Image limit for free community accounts

Step 3: Train Model

Roboflow applies data augmentation (rotation, brightness adjustment) and trains YOLOv8 on your dataset—no coding required

Step 4: Evaluate Performance

Review accuracy metrics, confusion matrix, and misclassified examples to understand model strengths and limitations

Step 5: Deploy and Test

Upload new MRI images to get instant classification predictions with confidence scores

Critical Step: Dataset Balancing

Why Balance Matters

Imbalanced datasets cause AI to favor common classes and ignore rare ones

Imbalanced Dataset Problem

369 Meningioma T1C+ images
18 Ganglioglioma T1C+ images
Result: Model predicts "Meningioma" for everything to maximize accuracy, missing rare tumors entirely

Balanced Dataset Solution

18 Images per class (use smallest class)
80% For training (14 images)
20% For validation/testing (4 images)
Result: Model learns all classes equally, improving rare tumor detection
