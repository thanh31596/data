CSS Animations Interactive Exercise

Instructions

Use the controls below to create your own CSS animations. Adjust the properties and see your animation in real-time!

CSS animations allow you to gradually change from one style to another without using JavaScript. You can control:

  • Animation Name: The name of the @keyframes animation
  • Duration: How long the animation takes to complete one cycle
  • Timing Function: How the animation progresses through time (ease, linear, etc.)
  • Delay: How long to wait before starting the animation
  • Iteration Count: How many times the animation should repeat
  • Direction: Whether the animation should alternate or reset and repeat
  • Fill Mode: What values are applied before/after the animation
