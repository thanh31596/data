Understanding Element, Class, and ID Selectors

Element Selectors

Element selectors target HTML elements directly by their tag name.

p { color: #666; }
h1 { color: #333; }
button { background-color: #4a6fa5; }

The above CSS rules apply to all instances of these elements on the page.

Class Selectors

Class selectors target elements with a specific class attribute. Classes can be reused across multiple elements.

This paragraph uses the "highlight" class to stand out from the rest.

This is a success message using the "success" class.

This is an error message using the "error" class.

.highlight { background-color: #ffffd9; }
.success { color: #155724; background-color: #d4edda; }
.error { color: #721c24; background-color: #f8d7da; }

ID Selectors

ID selectors target elements with a specific id attribute. IDs must be unique within a page.

Look at the #header, #main-content, and #footer sections of this page.

#header { background-color: #4a6fa5; color: white; }
#main-content { padding: 0 15px; }
#footer { text-align: center; background-color: #333; }

Selector Specificity

Selectors have different levels of specificity:

  1. ID selectors (most specific)
  2. Class selectors (medium specificity)
  3. Element selectors (least specific)

More specific selectors override less specific ones when styling the same property.

Combining Selectors

You can combine selectors to target specific elements within a context:

#main-content p { text-align: justify; }
.container h2 { border-bottom: 2px solid #eee; }
