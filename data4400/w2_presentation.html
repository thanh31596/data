Time Series Dataset Characteristics

For reliable time series analysis, data must have:

✓ Equal Time Increments

Regular intervals (daily, weekly, monthly, yearly)

Good: Jan, Feb, Mar, Apr...
Bad: Jan, Mar, May, Aug...

✓ No Missing Values

Complete data for all time periods

Good: 100, 120, 130, 125
Bad: 100, -, 130, 125

✓ Plausible Values

No implausible outliers or errors

Good: 100, 120, 130, 125
Bad: 100, 120, 9999, 125

✓ No Erroneous Values

Data entry errors corrected

Check: Negative sales impossible
Check: Values within expected range

Key Time Series Definitions

Period

The length of one complete cycle in a repeating pattern

Weekly sales: Period = 7 days

Frequency

How often an event occurs within a period

Monthly data: 12 observations per year

Realization

A single observed value from a time series

Sales in January: 150 units

Mean & Standard Deviation

Average value and measure of variability

Mean: Central tendency
Std Dev: How spread out the data is

Stationarity

When statistical properties don't change over time

Constant mean and variance

5 Components of Time Series

Time series can consist of all, or some of, these components:

1. Long-term Trend
2. Stationarity
3. Seasonality
4. Cyclical Pattern
5. Irregularity (Random Variation)
